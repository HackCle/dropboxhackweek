    Hey Dropbox, thanks

    Thanks for taking time to work on some important civic engagement and social justice projects! They will impact the city of Cleveland, Ohio and in the future, whatever city would like to replicate the projects. Your time is appreciated above measure.

    Clarity

    These projects came out of the FIX 216 Civic Hackathon and we've had a couple follow up meetings since the event to make sure all information is up-to-date and organized. If anything is unclear or if you have a question, ask us! Email seth@hackcleveland.org

    Project #2

    GOVERNMENT MEETING MINUTE ANALYSIS

    System to analyze meeting minutes currently locked in PDFz (currently for Cuyahoga County Council committees and subcommittees)

    At-a-glance users are able to identify if there were any absences and if there was an opportunity for public comment (indicated with Xs and check marks); relevant motions, which members were absent.

    How You Can Contribute

    Mine data in order to expand current data set from all of Cuyahoga County to City of Cleveland so users can filter results by city

    Project #3

    PUBLIC SERVANT INTERACTION API

    Database to track points of interface between the public and public servants (police, prosecutors, elected officials, building inspectors, etc.)

    Valuable resource for future app development; data is also customizable for maps, infographics, and reports. Could lead to ACLU-type app or Yelp for the police app.

    How You Can Contribute

    Create a landing page that links to the open source project and provides content that explains to city leadership the meaning and value of using APIs
